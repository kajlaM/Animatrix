Animatrix

Animatrix is a collection of AI-powered animation and visualization projects built using Python, Gemini API, and Manim.

The goal of this repository is to explore how large language models can generate mathematical and educational animations from natural language prompts.

⸻

Modules

AI Animation Pipeline

Generate Manim animations directly from natural language descriptions using Gemini.

Features

* Natural language → Manim code generation
* Automatic code cleaning and extraction
* Video rendering using Manim
* Example generated animations included
* Modular project structure for future extensions

Tech Stack

* Python
* Gemini API
* Manim Community Edition
* VS Code

⸻

Example Outputs

Circle to Square Animation

A simple demonstration of shape transformation using Manim.

Pythagorean Theorem Visualization

Animated geometric proof and labeling of a right triangle.

Fourier Series Animation

Visualization of how periodic functions can be approximated using harmonic components.
